# 火炬修仙转 — 游戏设计文档 (GDD)

> 单机 · 随机事件 · 选择驱动 · 火炬无限英雄 × 凡人修仙传境界  
> 项目代号：`tli-choose`  
> 文档版本：v0.5 · 2026-07-14  
> 已锁定：火炬暗金 · 一命 · 无轮回 · 英雄原名 · 仅中文 · 开局随机天赋 · 年历行动 · 渡劫终局  
> 新增：**主修功法（Active Skill）· 每境界阶段一席 · 功法等级 · 战斗/奇遇获取**

## 1. 一句话定位

**你是穿越到修仙界的火炬英灵。在随机的天南机缘中做出选择，以《凡人修仙传》式境界一步步飞升；每一局都是一次「机缘轮转」。**

| 维度 | 定义 |
|------|------|
| 类型 | 单机年历行动 + 事件选择 Roguelike（一命硬核） |
| 核心循环 | 选行动 → 耗时 → 修为/事件/战斗结算 → 进阶境界 → 直至渡劫 |
| IP 融合 | 火炬无限 **原名英雄 + 特性** 做道途；境界按 **凡人修仙传** |
| 美术 | **火炬暗金**（暗底、金描边、火炬式 UI，不做水墨皮） |
| 语言 | **仅中文**（不做 i18n / 英文包） |
| 天赋 | 开局从 [火炬天赋](https://tlidb.com/cn/Talent) 池 **随机抽取**，本局固定 |
| 功法 | 来自 [主动技能](https://tlidb.com/cn/Active_Skill)；**每大境界阶段 1 个主修**；可升级；战斗/奇遇获取 |
| 胜利条件 | 达到指定大境界（MVP：结丹/化神线；完整：大乘飞升）或完成主线机缘 |
| 失败条件 | 寿元耗尽、身死道消、心魔反噬等 → **本局结束，无轮回继承** |

## 2. 设计目标与非目标

### 2.1 目标

1. **选择有重量**：每个选项应改变至少一种「资源 / 关系 / 风险 / 境界进度」。
2. **英雄辨识度**：选不同火炬英雄，事件池权重、可用选项、战斗结算方式明显不同。
3. **境界沉浸感**：炼气十三层、筑基瓶颈、结丹、元婴分神……节奏像凡人，不做成纯数值升级。
4. **可重玩**：同一英雄不同「气运种子」+ **随机天赋盘** + 事件池，形成不同道途（**无永久元进度**）。
5. **一命紧张感**：死亡即终局；中途仅有「当前进行中」自动档，不可读档刷结果。
6. **单机可离线**：本地进行中存档，无强制联网。

### 2.2 非目标（明确不做）

- 实时动作 / ARPG 清怪（那是火炬本体）。
- 大型开放地图探索。
- 网游养成、交易行、赛季经济。
- 完整复刻凡人小说剧情（只借境界体系与修仙氛围）。
- **轮回 / 元进度永久加成**（图鉴解锁加成、轮回点、新局继承等一律不做）。
- **多档读档、手动存读刷结局**（违反一命）。
- 水墨/仙侠皮肤切换（美术统一火炬暗金）。
- **多语言 / i18n**（仅中文）。
- 完整复刻火炬天赋树点亮玩法（本作为**开局随机抽取**，非局内自由加点和洗点）。

## 3. 核心体验循环

进度以 **修仙年历** 推进：玩家在「本月/本季/本年」选择 **主动行动**（修炼、探险、寻人斗法…），行动 **消耗时间**，结算 **修为/资源/风险**，修为攒满后 **进阶境界**，一路从炼气走到 **大乘圆满 → 渡劫飞升**。

```
┌──────────────────────────────────────────────────────┐
│ 开局：选英雄 → 出身 → 随机天赋盘（锁定）              │
└────────────────────────┬─────────────────────────────┘
                         ▼
              ┌──────────────────────┐
              │  年历面板（主界面）   │  显示：年龄/寿元、境界、修为条、
              │  选择一项【行动】     │  当前年份、可用行动列表
              └──────────┬───────────┘
                         ▼
         扣除时间（月/年）→ 执行行动逻辑
                         ▼
         ┌───────────────┴───────────────┐
         ▼                               ▼
   直接数值结算              触发随机事件/斗法/奇遇
   （修为↑ 灵石± …）         （叙事 + 2~4 选项）
         └───────────────┬───────────────┘
                         ▼
              修为是否满？寿元是否尽？
                         ▼
         ┌───────────────┼───────────────┐
         ▼               ▼               ▼
    小层次进阶      大境界突破仪式     继续选下一项行动
    （自动/提示）    （天劫/瓶颈事件）
                         ▼
              大乘圆满 → 渡劫期 → 飞升/身死
```

**一局时长目标**  
| 阶段 | 境界目标 | 现实时长 | 游戏内年岁跨度（约） |
|------|----------|----------|----------------------|
| MVP | 炼气 → 结丹 | 20–40 分钟 | 数十～百余年 |
| 标准 | → 化神 | 45–75 分钟 | 数百年 |
| 完整 | → **渡劫飞升** | 90–150 分钟 | 上千年（靠闭关压缩操作） |

## 4. 境界体系（凡人修仙传）

对齐《凡人修仙传》大框架：**下境界 → 中境界 → 渡劫飞升（入仙）**。  
「渡劫」在原著中是大乘之后升仙的天劫过程；本作将其做成可游玩的终局阶段 **渡劫期**（筹备 + 九重劫 + 飞升检定）。

### 4.1 大境界总表

| 阶 | 大境界 | 细分 | 寿元上限（设计用） | 游戏定位 |
|----|--------|------|--------------------|----------|
| 下境界 | **炼气** | **1–13 层** | ~100 岁 | 新手、资源极紧 |
| 下境界 | **筑基** | 初 / 中 / 后 / 大圆满 | ~200–250 岁 | 真正入道 |
| 下境界 | **结丹** | 初 / 中 / 后 / 大圆满 | ~500 岁 | 中期核心（MVP 首通目标） |
| 下境界 | **元婴** | 初 / 中 / 后 / 大圆满 | ~1000 岁 | 一方修士 |
| 下境界 | **化神** | 初 / 中 / 后 / 大圆满 | ~2000+ 岁 | 下境界顶峰 |
| 中境界 | **炼虚** | 初 / 中 / 后 / 大圆满 | 极长 | 完整版后期 |
| 中境界 | **合体** | 初 / 中 / 后 / 大圆满 | 极长 | 完整版后期 |
| 中境界 | **大乘** | 初 / 中 / 后 / **大圆满** | 近飞升 | 终局前最后修为期 |
| **终局** | **渡劫期** | 筹备 / 渡劫中 / 劫后 | 劫中不论寿 | **通关阶段** |
| 上境界 | **真仙 → 金仙 → 太乙 → 大罗 → 道祖** | 各层次 | 仙寿 | 通关后扩展（可选） |

**通关条件（完整版）**：大乘大圆满 → 进入渡劫期 → 度过飞升天劫 → **飞升真仙**。  
**MVP 通关**：结丹成功（或可选挑战元婴）；身死/寿尽仍算一命终局。

### 4.2 小层次与修为条

| 境界类型 | 小层次 | 每层修为需求 |
|----------|--------|--------------|
| 炼气 | 第 1～13 层 | 每层 `cultivation` 0→100，满则升一层 |
| 筑基及以后（至大乘） | 初期→中期→后期→大圆满 | 每小层次 0→100；满且满足条件才可破大境界 |
| 渡劫期 | 见 §4.6 | 不走常规修为条，走 **劫雷进度 / 筹备度** |

修为单位游戏内可显示为「修为点」或百分比条；底层用整数便于计算。

### 4.3 炼气十三层（教学节奏）

| 层 | 内容侧重 | 典型可用行动 |
|----|----------|--------------|
| 1–3 | 学会年历、修炼与坊市 | 闭关、打杂赚钱、低危探索 |
| 4–7 | 散修斗法、低级秘境 | 探险、寻仇/切磋解锁 |
| 8–12 | 囤筑基资源 | 寻宝、交易、炼丹、拜山 |
| **13** | **筑基瓶颈** | 强制/推荐进入「筑基」行动链 |

### 4.4 各大境界突破条件（凡人风）

| 突破 | 修为条件 | 关键资源/状态（示例） | 天劫/风险 |
|------|----------|------------------------|-----------|
| 炼气→**筑基** | 炼气 13 层满 | 筑基丹（或邪功/机缘替代）、道心≥30 | 筑基雷劫（轻） |
| 筑基→**结丹** | 筑基大圆满 | 结丹材料、足够灵力环境 | 结丹心魔+雷劫 |
| 结丹→**元婴** | 结丹大圆满 | 婴变机缘/宝药、神识 | 婴变劫，失败重创 |
| 元婴→**化神** | 元婴大圆满 | 化神之道、天地异象 | 化神劫 |
| 化神→**炼虚** | 化神大圆满 | 空间感悟、虚境机缘 | 炼虚劫 |
| 炼虚→**合体** | 炼虚大圆满 | 法则碎片、道侣/本命物等叙事 | 合体劫 |
| 合体→**大乘** | 合体大圆满 | 大乘机缘 | 大乘劫 |
| 大乘→**渡劫期** | 大乘大圆满 | 主动选择「准备飞升」 | 进入终局阶段 |
| 渡劫→**真仙** | 渡劫成功 | 抗劫法宝、道心、功德/杀孽影响 | **飞升天劫**（九重） |

### 4.5 突破结算模板（行动：突破）

玩家在修为满且材料齐备时，年历上出现 **【突破】**（或瓶颈强制提示）：

1. **准备**：吃药 / 布阵 / 找护法 / 邪功捷径（选项）。  
2. **天劫事件链**：多段检定（体/志/运 + 天赋 + 法宝）。  
3. **结果**：  
   - 成功 → 晋升下一境初期，寿元上限重置为该境表，部分伤势清除。  
   - 失败 → 修为倒退 / 重伤 / 走火入魔 / **身死**（一命结束）。  
   - 放弃 → 可继续修炼但卡在圆满，**空耗寿元**。

### 4.6 渡劫期（终局专章）

大乘大圆满后解锁行动 **「引动飞升天劫」** / **「渡劫筹备」**。

| 子阶段 | 玩家在做什么 | 时间消耗 | 失败 |
|--------|--------------|----------|------|
| **筹备** | 寻抗劫宝、稳道心、清因果、布置劫阵 | 数年～数十年 | 被仇家突袭、心魔 |
| **渡劫中** | 九重天劫，每重一次检定/事件 | 以「劫」计时，现实数个决策 | 任一重致命失败→身死 |
| **劫后** | 飞升门开，选择入仙或滞留（滞留无通关） | 短 | — |

- 杀孽过高：劫力增强。  
- 功德/气运：减伤或额外容错。  
- 英雄/天赋：如「坚强」可把首次致死改为重伤（每局限 1）。  
- **渡劫成功 = 本局胜利（飞升）**。

### 4.7 修为从哪里来（与行动挂钩）

| 来源 | 稳定性 | 风险 | 典型行动 |
|------|--------|------|----------|
| 闭关修炼 | 高 | 低（走火低概率） | 修炼 |
| 探险历练 | 中 | 中高 | 探险 |
| 斗法/PK | 中低 | 高 | 寻人交手 |
| 丹药服食 | 爆发 | 耐药、反噬 | 炼丹/购丹后「服药」 |
| 机缘顿悟 | 极高 | 看事件 | 探险/奇遇内选项 |
| 宗门任务/打杂 | 低 | 低 | 事务 |
| 天赋/功法被动 | 常驻修正 | — | 乘区，不单独耗时 |

## 5. 年历与行动系统（进度核心）

### 5.1 时间单位

| 单位 | 定义 | 用途 |
|------|------|------|
| **年** | 主进度显示（第 N 年 / 虚岁） | 寿元、总结、时代感 |
| **月** | 行动耗时的基本粒度（1 年 = 12 月） | 绝大多数行动扣月 |
| 日/时辰 | 仅剧情内嵌，不单独做操作层 | 天劫、短事件描写 |

- 开局年龄：由出身决定（如 16～25 岁）。  
- `age` 随扣除的月份累计：`age += months / 12`（可显示到月或只显示年）。  
- `age >= lifespan` → **寿元耗尽**，一命结束（可有「强撑一年」类绝境事件，默认无）。

### 5.2 主界面：行动菜单

玩家不「纯抽卡过日子」，而是 **点行动 → 耗时 → 出结果**。  
随机事件是行动的结果或插曲，不是唯一循环。

**行动列表（按解锁逐步出现）：**

| 行动 ID | 名称 | 耗时（默认） | 主要收益 | 主要风险 | 解锁 |
|---------|------|--------------|----------|----------|------|
| `cultivate` | **修炼/闭关** | 3 / 6 / 12 月（自选） | 修为（稳定） | 走火、心魔（低） | 开局 |
| `cultivate_art` | **专修功法** | 3 / 6 月 | 当前主修 **技能等级**↑ | 走火（极低） | 已装备主修 |
| `explore` | **探险/历练** | 1～6 月 | 修为、材料、奇遇事件 | 遇敌、重伤、身死 | 开局 |
| `duel` | **寻人交手 (PK)** | 1～3 月 | 修为、声望、战利品、杀孽 | 战败重伤/身死 | 炼气 4+ 或结仇后 |
| `travel` | **游历/赶路** | 1～12 月 | 切换区域、见闻 | 路遇 | 开局 |
| `trade` | **坊市交易** | 1 月 | 买卖、情报 | 被坑、被盯上 | 到达坊市 |
| `alchemy` | **炼丹** | 1～3 月 | 丹药 | 炸炉、损材 | 配方+境界 |
| `craft` | **炼器/布阵** | 2～4 月 | 法器、阵盘 | 失败损材 | 中期 |
| `sect` | **宗门/势力事务** | 1～6 月 | 贡献、资源、关系 | 卷入斗争 | 加入势力后 |
| `social` | **访友/拜山/缔盟** | 1～3 月 | 关系、隐藏机缘 | 得罪人 | 炼气后 |
| `heal` | **疗伤/静养** | 1～6 月 | 回 hp、去伤势 | 耗时、可能被袭 | 有伤时 |
| `breakthrough` | **突破境界** | 1～3 月（仪式） | 晋升 | 天劫失败身死 | 修为满+材料 |
| `tribulation` | **渡劫/飞升** | 特殊 | 通关 | 九死一生 | 大乘圆满 |

> 耗时可被天赋/环境修正：如狩猎之神「行色匆匆」→ 探险耗时 -1 月（最少 1）；知识之神闭关效率 +%。

### 5.3 行动详述

#### 5.3.1 修炼 / 闭关（`cultivate`）

玩家选择时长档：

| 档位 | 耗时 | 基础修为 | 备注 |
|------|------|----------|------|
| 短关 | 3 月 | 低 | 灵活，适合卡寿元前微调 |
| 年关 | 6 月 | 中 | 默认推荐 |
| 苦修 | 12 月 | 高 | 可能触发闭关事件（顿悟/走火/被挖洞） |

```
修为增益 ≈ base(档位) × 灵根修正 × 环境(灵脉) × 功法 × 天赋 × (1 - 伤势惩罚)
```

- **灵根**：优 → 乘区高；伪灵根/杂灵根低但可有奇遇补偿。  
- **环境**：洞府灵脉等级 0～5；游历到洞天可临时加成。  
- **耐药**：连续苦修不活动会提升「心魔权重」，需穿插历练/访友。  
- 闭关中若世界事件（仇家寻上门）可被打断 → 插入 PK/逃亡事件。

#### 5.3.2 探险 / 历练（`explore`）

1. 选 **目标类型**（随境界解锁）：山野 / 坊市暗巷 / 秘境入口 / 古修洞府 / 海域…  
2. 选 **风险倾向**：谨慎（耗时长、收益稳）/ 寻常 / 冒进（耗时短或收益高、死亡率高）。  
3. 扣时间 → 抽 **探险事件组**（1～3 个节点）→ 选项结算。  
4. 结算包：修为 + 物品 ± hp + 可能解锁地图。

探险是 **内容消费主入口**，也是材料与机缘主要来源。

#### 5.3.3 寻人交手 / PK（`duel`）

| 子类型 | 说明 |
|--------|------|
| 切磋 | 不死，修为与声望小增，可结善缘 |
| 生死斗 | 可杀可夺宝，杀孽↑，败则重伤或身死 |
| 寻仇 | 关系网触发，对方可能叫人 |
| 猎杀悬赏 | 任务型，境界压制影响胜率 |

流程：选目标（列表随见闻生成）→ 耗时寻访 → 进入 **战斗子系统** → 结果回写年历。

境界压制（凡人味）：

- 高一小层次：小优  
- 高一大境界：显著优势（低境硬刚成功率极低，除非奇谋/宝药/英雄技）

#### 5.3.4 其他行动（摘要）

| 行动 | 设计要点 |
|------|----------|
| 游历 | 改变 `region`，切换事件池与灵脉；长途高耗时 |
| 交易 | 不涨多少修为，服务经济；可遇骗子事件 |
| 炼丹 | 赛琪等英雄加成；丹药供突破与探险 |
| 事务 | 稳定灵石/贡献，修为少；可能升长老线 |
| 访友 | 降心魔、接任务、道侣/道友线、隐藏机缘 |
| 疗伤 | 伤势不处理则修炼效率↓、突破成功率↓ |
| 突破/渡劫 | 见 §4.5、§4.6 |

### 5.4 一年可以怎么过（节奏示例）

**炼气期某一年（操作 3～6 次）：**

| 顺序 | 行动 | 耗时 | 结果意象 |
|------|------|------|----------|
| 1 | 闭关短关 | 3 月 | 修为 + |
| 2 | 坊市交易 | 1 月 | 买回气丹 |
| 3 | 探险山野 | 2 月 | 遇狼群/散修，得草药 |
| 4 | 闭关年关 | 6 月 | 修为接近升层 |

**筑基后某十年（可合并显示）：**  
系统允许「计划十年：6 年闭关 + 2 年历练 + …」一类 **宏行动**（完整版），避免后期每年点 12 次；MVP 用 3/6/12 月档即可。

### 5.5 时间 vs 收益 vs 寿命（核心张力）

```
想快突破 → 冒进探险/邪功/连服丹药 → 风险↑、杀孽/耐药↑
想稳 → 长期闭关 → 安全但可能机缘不够、卡材料、心魔
拖太久 → 寿元见底 → 必须赌突破或找延寿
```

延寿来源（稀有）：宝药、特殊功法、邪功吸寿（杀孽）、机缘。  
**不提供**轮回继承；延寿只作用于本局。

### 5.6 修为需求曲线（设计用基准）

数值可调，原则：**同境越高层越慢；大境界之间有平台感**。

| 阶段 | 单层/小层次满条参考「基础修为点」 | 建议主手段 |
|------|-----------------------------------|------------|
| 炼气 1–5 | 80～120 | 闭关为主 |
| 炼气 6–13 | 120～200 | 闭关+探险 |
| 筑基各阶 | 300～500 | 灵脉洞府+历练 |
| 结丹各阶 | 600～1000 | 秘境+丹药 |
| 元婴各阶 | 1200～2000 | 势力冲突+大机缘 |
| 化神～合体 | 更高 | 法则/虚境向事件 |

**单次行动修为**应让玩家在炼气期 **2～4 次行动升 1 层**，避免纯枯坐；筑基后变慢。

### 6.2 六维战斗/检定属性（简化）

| 属性 | 对应火炬感 | 修仙解释 |
|------|------------|----------|

## 8. 天赋系统（火炬天赋 · 开局随机）

数据与命名来源：[tlidb.com/cn/Talent](https://tlidb.com/cn/Talent)  
火炬原版结构约：**神系/传承 Profession ~31**、**核心天赋点 ~138**、**天赋节点 ~1020**（含小型/中型/传奇中型）。

本游戏**不复刻**「局内走天赋树加点」，而是把天赋当作 **本命灵纹 / 天道烙印**：开局随机生成一套，**整局锁定**（一命、无洗点、无轮回继承）。

### 8.1 设计目标

1. **每局 Build 手感不同**：同英雄不同天赋 → 事件选项与检定倾向变化。  
2. **保留火炬辨识度**：神系名、核心天赋名、效果文案尽量用 wiki 原名与原意。  
3. **适配文字选择玩法**：把 ARPG 秒伤词条 **映射** 为修仙数值 / 标签 / 专属选项，而非 1:1 战斗帧。  
4. **可落地**：MVP 先做「可映射子集」，再逐步扩到全量 1000+ 节点。

### 8.2 开局抽取规则（本命天赋盘）

选完英雄与出身后，进入 **「天道授纹」** 动画（暗金 UI），按种子 `seed` 随机：

| 槽位 | 数量 | 池子 | 说明 |
|------|------|------|------|
| **神系/传承** | 1 | Profession（六神 + 各传承/侍者，含新神为稀有） | 决定主属性倾向与事件标签权重 |
| **核心天赋** | 1（困难可选 2） | 核心天赋点，**优先同神系/同传承** | 本局「大词条」机制，可解锁专属选项 |
| **灵纹天赋** | 3 | 小型 + 中型 + 传奇中型节点 | 数值向修正；传奇中型权重低 |

**约束：**

- 同一 `talentId` 不重复。  
- 核心天赋默认从 **当前神系/传承所属** 池抽取；若池为空则回退到该神系任意核心。  
- **新神** 神系权重极低（如 3%），抽中则核心从新神 6 核中抽。  
- **不可重掷**（服务一命与种子可复现）；标题「开始」前若取消选人则未生成。  
- 展示：神系徽章 + 核心天赋大图标 + 3 条灵纹摘要；确认后进入第一事件。

```
权重示例（可调）：
  神系：六神各均等；新神 3%
  核心：所属传承 70% / 同神其他传承 25% / 跨神 5%（跨神更混沌）
  灵纹：小型 55% / 中型 35% / 传奇中型 10%
```

### 8.3 神系 → 修仙映射

| 火炬神系/方向 | 主属性 | 修仙叙事 | 事件权重↑ | 六维倾向 |
|---------------|--------|----------|-----------|----------|
| **巨力之神** | 力量 | 刚体霸道、肉身成圣倾向 | 斗法、硬闯、力破 | 攻、体 |
| **狩猎之神** | 敏捷 | 风遁猎手、以快破敌 | 潜行、逃亡、猎妖 | 敏、运 |
| **知识之神** | 智慧 | 神识、阵法、术法研究 | 闭关、残卷、秘术 | 智、志 |
| **征战之神** | 力+敏 | 杀伐果决、战场气运 | 战争、比斗、护道 | 攻、敏 |
| **欺诈之神** | 敏+智 | 心魔、诅咒、以毒攻毒 | 魔道、交易欺诈、阴诡 | 智、运（道心风险↑） |

- 核心天赋抽取池  
- 少量专属事件（如「战吼」类 → 示威/聚势选项；「哨卫」类 → 布置禁制）

### 8.4 效果如何生效（映射层）

火炬原文保留展示；运行时走 **`TalentEffect` 统一效果表**：

```ts
type TalentEffect =
  | { type: 'attr'; key: AttrKey; value: number }           // 攻敏智体运志
  | { type: 'resource'; key: 'maxHp'|'maxMana'|...; pct?: number; flat?: number }
  | { type: 'checkBonus'; tags: string[]; value: number }  // 某类检定 +%
  | { type: 'eventWeight'; tags: string[]; mult: number }  // 事件池权重
  | { type: 'unlockChoice'; choiceFlag: string }           // 解锁带 flag 的选项
  | { type: 'combat'; key: string; value: number }         // 战斗子系统
  | { type: 'rule'; id: string; params?: Record<string, number> } // 特殊规则
  | { type: 'curse'; id: string }                          // 负面烙印（高风险核）
```

**映射原则（举例）：**

| 火炬效果（意） | 本游戏效果 |
|----------------|------------|
| +伤害 / +攻击伤害 | 攻 +N 或战斗伤害修正 |
| +最大生命 / 护甲 | 体、maxHp |
| +攻击速度 / 移动 | 敏；「速决」类选项耗时减少 |
| +闪避 / 偏斜 | 逃跑、躲避检定 + |
| +法术伤害 / 聚能 | 智；术法选项 |
| 坚韧祝福相关 | 规则：战斗/硬抗选项叠「坚韧」层 |
| 灵动祝福相关 | 规则：连续「移动/猎杀」选项叠「灵动」 |
| 淘汰（残血斩杀） | 战斗：敌方 hp&lt;18% 时额外处决检定 |
| 坚强（保留 1 命） | **一命例外？** → 映射为「首次致死改为濒死，仅 1 次/局」（保留紧张感） |
| 世事无常（高低伤） | 运相关：检定结果更极端（大成功/大失败权重↑） |
| 禁忌力量（+腐蚀 -抗性） | 魔道事件权重↑，正道检定难度↑ |
| 号令/哨卫/智械 | 召唤物槽或「布置机关」选项 |
| 新神·吞噬祝福 | 消耗对应层数换一次爆发选项 |

无法合理映射的纯 ARPG 帧同步效果：降级为 **近似数值包** 或 MVP **暂不入抽取池**（`enabled: false`）。

### 8.5 内容数据结构

```ts
// content/talents/*.json — 与 tlidb 对齐的中文原名
interface TalentDef {
  id: string;                 // 建议用 wiki data-id，如 "20001100"
  name: string;               // 「淘汰」「借势」「小型天赋」…
  kind: 'core' | 'notable' | 'small' | 'medium' | 'legendary_medium';
  profession: string;         // 「巨力之神」「勇者」…
  professionId: string;       // God_of_Might / The_Brave
  description: string;        // 火炬原文（中文）
  icon?: string;
  effects: TalentEffect[];    // 游戏映射
  tags: string[];             // attack, spell, minion, ailment...
  weight?: number;            // 抽取权重覆盖
  enabled: boolean;           // MVP 可先 false 大量节点
}

interface TalentLoadout {
  professionId: string;
  professionName: string;
  core: TalentDef;
  nodes: TalentDef[];         // 3 个灵纹
}
```

| **Alpha** | 核心天赋点扩到 ~80；传承名展示；传奇中型入池 |
| **1.0** | 尽量覆盖 wiki 核心全量 + 常用灵纹；剩余小型用模板批量映射（+伤害→攻 等） |

> 1020 节点不必手写每一条特殊逻辑：小型/中型用 **词条模板解析**（正则/规则表把「+9% 伤害」→ effects），核心与传奇中型手写 `rule`。

### 8.8 开局流程（更新）

1. 标题 → 新开一局（生成 `seed`）  
2. 选英雄（原名）+ 特性分支  
3. 选出身  
4. **天道授纹**：展示随机天赋盘（不可改）→ 确认  
5. **立炼气主修**：从赠送/已悟功法中选 1 部  
6. 进入年历主界面（行动循环）

## 9. 功法系统（火炬主动技能 → 主修功法）

数据与命名来源：[tlidb.com/cn/Active_Skill](https://tlidb.com/cn/Active_Skill)（约 197 个主动技能，含 Tag：力量/敏捷/智慧、物理/火焰/冰冷/闪电/腐蚀、攻击/法术/位移/召唤…）。

本作把主动技能转译为 **功法**（原名保留，如「跃击」「裂变火球」「闪电链」）。玩家通过战斗、奇遇等 **收录功法**，并在每个 **大境界阶段** 选定 **1 部主修功法**；功法有 **品阶 + 技能等级**，决定闭关效率、斗法选项与事件分支。

### 9.1 设计目标

1. **阶段 Build**：炼气、筑基、结丹…各自有主修，风格可随境界转型（近战→法术亦可）。  
2. **收集驱动**：探险/PK 有明确掉落期待（残篇、全本、同名升级）。  
3. **火炬辨识度**：技能中文原名 + Tag 体系，暗金图标框展示。  
4. **服务文字玩法**：不做成 ARPG 连招帧，而映射为选项、检定、修为乘区。

### 9.2 概念分层

| 概念 | 含义 |
|------|------|
| **功法图鉴 / 已悟** | 本局已学会的功法列表（可多部） |
| **主修功法** | 当前大境界装备的那 1 部，效果全额生效 |
| **往境烙印** | 过往大境界的主修，保留 **削弱生效**（默认 40% 效果） |
| **辅修（可选）** | MVP 不做；完整版每境可再挂 1 个辅修（20% 效果） |
| **品阶** | 功法「品质」：凡 → 黄 → 玄 → 地 → 天 → 仙 |
| **技能等级** | 1～10（MVP）；完整版可扩到 1～20，对齐火炬技能等级感 |
| **Tag** | 继承火炬标签，用于与天赋/英雄/事件匹配 |

### 9.3 阶段主修槽（每境一个）

按凡人修仙传 **大境界** 开槽（进入该境后可装备/更换）：

| 阶段槽 | 解锁时机 | 备注 |
|--------|----------|------|
| 炼气主修 | 开局（或炼气 3 层教学后） | 开局赠 1 部基础功法 |
| 筑基主修 | 筑基成功后 | 可与炼气主修相同或更换 |
| 结丹主修 | 结丹成功后 | |
| 元婴主修 | 元婴成功后 | |
| 化神主修 | 化神成功后 | |
| 炼虚 / 合体 / 大乘主修 | 对应突破后 | 完整版 |
| 渡劫主修 | 进入渡劫期 | 影响抗劫检定与飞升选项 |

**规则：**

1. **每个阶段有且仅有 1 个主修席**。  
2. 新晋大境界时：弹出 **「立主修」**——从已悟功法中选 1 部装入本境槽（可与上境相同）。  
3. **更换主修**：年历行动「更换主修」耗时 1～3 月 + 少量灵石/心魔风险；一境建议限制次数（如每境 2 次），防反复刷最优。  
4. **仅当前境主修**享受 100% 效果与专属战斗选项；往境主修转为烙印（见 9.2）。  
5. 未装主修时：闭关效率惩罚、斗法仅能用凡俗拳脚（极弱）——逼玩家尽早立修。

### 9.4 品阶与技能等级

#### 9.4.1 品阶（获取时判定，可再升）

| 品阶 | 约略占比 | 基础强度 | 常见来源 |
|------|----------|----------|----------|
| 凡 | 开局/杂鱼 | 低 | 出身、路人 |
| 黄 | 高 | 标准 | 常规探险、低阶 PK |
| 玄 | 中 | 强 | 秘境、精英 |
| 地 | 低 | 很强 | 高阶洞府、元婴战 |
| 天 | 极低 | 极强 | 大机缘、化神+ |
| 仙 | 传说 | 破格 | 渡劫相关、唯一事件 |

- 同一技能名可因获取方式不同而有不同 **初始品阶**（如坊市伪本=黄，真传=玄）。  
- **合书/残篇**：两本同名可合成升品或补全效果。  
- 品阶影响：效果系数、可升技能等级上限（凡≤4，黄≤6，玄≤8，地≤10，天/仙≤10 且自带额外 rule）。

#### 9.4.2 技能等级（养成）

| 等级 | 需求（示意） | 效果 |
|------|--------------|------|
| 1 | 初学 | 解锁基础选项/最低乘区 |
| 2–4 | 专修行动 / 实战熟练 | 数值递增强 |
| 5 | 小成 | 解锁进阶选项或战斗形态 |
| 6–9 | 持续专修+战斗 | 接近圆满 |
| 10 | 大成 | 满效果；突破下境时有额外成功率或专属剧情 |

```
升级进度 skillXp：
  专修功法行动 → 大量
  使用该功法战斗胜利 → 中量
  探险中选了功法相关选项 → 少量
  与主修 Tag 匹配的闭关 → 附带微量 skillXp
```

**专修功法**行动（`cultivate_art`）：

- 耗时 3/6 月；主要涨 **当前主修** 的 skillXp，少量涨境界修为。  
- 与「闭关涨境界修为」分工：纯升境闭关 vs 磨刀专修。

### 9.5 功法效果如何生效

与天赋类似，走映射表（保留火炬原文描述供展示）：

```ts
type ArtEffect =
  | { type: 'attr'; key: AttrKey; value: number }
  | { type: 'cultivateMult'; value: number }      // 闭关修为乘区
  | { type: 'combat'; key: string; value: number }
  | { type: 'checkBonus'; tags: string[]; value: number }
  | { type: 'unlockChoice'; choiceFlag: string }
  | { type: 'eventWeight'; tags: string[]; mult: number }
  | { type: 'rule'; id: string; params?: Record<string, number> }
```

**强度缩放：**

```
最终效果 = 基础effects × 品阶系数 × 等级曲线(level) × 槽位系数(主修1.0 / 烙印0.4)
```

**Tag 协同（重要）：**

| 情况 | 加成 |
|------|------|
| 主修 Tag ∩ 天赋神系倾向 | 闭关/战斗额外小幅加成 |
| 主修 Tag ∩ 英雄气质 | 如雷恩+攻击/近战功法 → 怒气选项更强 |
| 主修 火焰 且区域为火脉 | 修炼乘区↑ |
| 主修 腐蚀 且正道宗门 | 可能触发排斥事件 |

**映射举例（主动技能 → 修仙）：**

| 火炬技能 | Tag 向 | 游戏效果摘要 |
|----------|--------|----------------|
| 跃击 | 位移/近战/力量 | 探险「强突」选项；战斗破阵 |
| 旋风斩 | 近战/引导 | 群战优势；耗时稍长的连斩检定 |
| 裂变火球 | 法术/火焰 | 术法检定；灼烧持续 |
| 闪电链 | 法术/闪电 | 连锁伤多个敌人（多目标事件） |
| 闪现 | 位移 | 逃跑/穿阵成功率↑ |
| 石肤术 | 防护 | 减伤、硬抗天劫小幅 |
| 寒冰射击 | 远程/冰冷 | 远程 PK、风筝选项 |
| 召唤机械警卫 | 召唤/智械 | 召唤物槽（莫托向更强） |
| 诅咒类（易伤等） | 诅咒 | 削弱敌方检定；魔道权重 |
| 灵药/滴露 | 灵药 | 赛琪向；战斗前 buff 选项 |
| 战吼类 | 战吼 | 聚势、群buff、吓退散修 |

MVP 不实现全部 197：先做 **40 部** 高频技能的完整映射，其余 `enabled:false` 或仅作掉落收藏（只加图鉴小 flavor）。

### 9.6 获取途径

| 途径 | 说明 | 典型品阶 |
|------|------|----------|
| **开局** | 英雄/出身各赠 1 部基础（凡～黄） | 凡/黄 |
| **战斗 / PK 胜利** | 按敌人境界与类型掉落残篇/全本；精英保底黄+ | 黄～地 |
| **探险奇遇** | 古洞玉简、残碑、前辈遗蜕 | 黄～天 |
| **坊市 / 拍卖** | 花钱买；有伪本风险（名实不符） | 凡～玄 |
| **宗门授法** | 贡献兑换；受声望与职位限制 | 黄～玄 |
| **访友 / 拜山** | 道友传功、比试赌注 | 黄～地 |
| **闭关顿悟** | 极低概率自创/领悟已有残篇补全 | 不定 |
| **突破馈赠** | 破境成功有时天道补一部适配功法 | 黄～玄 |
| **渡劫机缘** | 终局向仙品线索 | 天/仙 |

**掉落规则要点：**

- 掉落池按敌人 `realm` + `tags` 加权（火属性妖兽更易掉火焰功法）。  
- **重复获得同名**：转化为 skillXp 或升品材料，不占重复格子。  
- **残篇**：已悟但不完整，效果 50%，集齐可「参悟合书」行动（耗时 1～3 月）。

### 9.7 数据结构

```ts
interface ArtDef {
  id: string;                 // 如 leap_attack / wiki 英文 slug
  name: string;               // 「跃击」火炬中文原名
  sourceUrl?: string;         // tlidb Active_Skill
  tags: string[];             // ["位移","攻击","近战","物理","力量"]
  description: string;        // 原文精简版（中文）
  baseGrade: ArtGrade;        // 建议默认品阶下限
  maxSkillLevel: number;      // 由品阶限制可覆盖
  minRealm?: MajorRealm;      // 修炼/装备门槛（如地阶需筑基+）
  effects: ArtEffect[];       // 1 级基准
  levelScaling?: ScalingRule; // 每级如何放大
  enabled: boolean;
}

type ArtGrade = 'mortal' | 'yellow' | 'mysterious' | 'earth' | 'heaven' | 'immortal'; // 凡黄玄地天仙

interface OwnedArt {
  artId: string;
  grade: ArtGrade;
  skillLevel: number;         // 1..10
  skillXp: number;
  complete: boolean;          // 是否非残篇
}

interface MainArtSlots {
  // key = major realm id
  qi_refining?: string;       // artId
  foundation?: string;
  core_formation?: string;
  // ...
  tribulation?: string;
}
```

玩家状态增补：`arts: OwnedArt[]`、`mainArtByRealm: MainArtSlots`。

### 9.8 与年历 / 战斗 / 突破的交互

| 系统 | 交互 |
|------|------|
| 闭关 | 主修提供 `cultivateMult`；Tag 匹配灵脉再加成 |
| 专修功法 | 专涨 skillLevel |
| 探险 | 主修解锁特有选项（如有闪现→「瞬移入洞」） |
| PK | 战斗行动栏带出主修战技；等级影响命中与伤 |
| 突破天劫 | 主修防护/体类 Tag 提高抗劫；战吼/激发类影响心魔检定 |
| 渡劫 | 渡劫主修决定飞升叙事分支与部分检定 |
| 天赋 | 神系与功法 Tag 协同；冲突不禁止但无协同 |

### 9.9 开局与前 30 年节奏（示例）

1. 开局：出身赠 **黄阶「石肤术」** 或英雄倾向功法（雷恩偏「专注斩/跃击」）。  
2. 炼气 1–5：立炼气主修；靠闭关+低危探险。  
3. 炼气 6–10：PK/秘境掉 **「裂变火球」残篇**，合书后可考虑换主修。  
4. 筑基成功：弹出筑基主修；可选保留近战或转元素法。  
5. 此后每破一大境重复「立主修 → 收集更高品阶 → 专修满级」。

### 9.10 内容分期

| 阶段 | 功法数量 | 目标 |
|------|----------|------|
| **MVP** | ~40 部 enabled + 映射 | 炼气→结丹；获取+主修+升级闭环 |
| **Alpha** | ~100 | 品阶合成、残篇、宗门兑换 |
| **1.0** | 接近 wiki 主动技能全表 | 收藏向；未精做映射的给通用模板效果 |
| **终局包** | 仙品/渡劫向 | 飞升专用 |

### 9.11 UI 要点

- 年历顶栏：`主修：跃击 · 玄阶 · Lv6`  
- 【功法】页：已悟列表 / 各境主修槽 / 升级进度  
- 破境成功插页：选择本境主修（暗金大卡，展示 Tag 与效果）  
- 战斗：选项名带功法名「以跃击突进」

### 9.12 与「技能等级」火炬设定的关系

火炬主动技能有 1～40+ 等级数值曲线。本作 **压缩为 1～10**：

- 展示可写「功法境界：第 N 重」以修仙口吻；  
- 内部仍 `skillLevel`；  
- 不需要玩家点 40 次升级。

## 10. 事件与选择系统（游戏主体）

### 10.1 事件数据结构

```ts
interface GameEvent {
  id: string;
  title: string;
  body: string;                 // 叙事
  tags: string[];               // ["combat","sect","demon","alchemy","hero:rehan"]
  minRealm: Realm;              // 最低境界
  maxRealm?: Realm;
  weight: number;               // 基础权重
  conditions?: Condition[];     // 声望/物品/道心/英雄
  choices: Choice[];
  once?: boolean;               // 本局仅一次
  chain?: string;               // 事件链 ID
}

interface Choice {
  id: string;
  text: string;
  hint?: string;                // 「需筑基」「凶险」
  requirements?: Requirement[]; // 属性/物品/英雄特性
  cost?: ResourceCost;          // 灵石、时日、矢、怒等
  outcomes: WeightedOutcome[];  // 多结果权重
  heroBonus?: HeroId[];         // 该英雄额外成功率或独有分支
}
```

### 10.2 事件分类

| 类型 | 占比(目标) | 例子 |
|------|------------|------|
| 日常修炼 | 20% | 闭关、采气、整理功法 |
| 奇遇机缘 | 15% | 古洞、残阵、天材地宝 |
| 人物交互 | 15% | 同道、魔修、仙子、奸商 |
| 势力与任务 | 15% | 宗门任务、赏金、护送 |
| 战斗斗法 | 15% | 截杀、秘境、比斗 |
| 生产养成 | 10% | 炼丹、炼器、种灵药 |
| 危机灾劫 | 5% | 心魔、围杀、天劫、寿元预警 |
| 英雄专属 | 5%+加权 | 见 §6 |

### 10.3 选项设计原则

1. **至少两难**：稳 vs 险，正 vs 邪，利已 vs 利人。  
2. **失败也可玩**：失败给叙事与状态，而非纯 Game Over（除非致命）。  
3. **英雄特权选项**：灰色锁定显示「需·怒火雷恩」，强化收集欲。  
4. **信息不全**：部分结果隐藏，靠「神识检定」或尤加时空预览。  
5. **链式后果**：救下某人 → 10 年后筑基时来报；结仇 → 元婴期追杀。

### 10.4 事件链示例：「筑基之夜」

1. 炼气 13 层触发「筑基机缘」总事件。  
2. 分支：  
   - A 坊市购伪丹（便宜，失败率高）  
   - B 宗门试炼换真丹（耗时、耗声望）  
   - C 魔修血祭（杀孽↑，成功率高，道心↓）  
   - D 雷恩专属：以怒火硬抗雷劫（体/志检定）  
3. 突破结算 → 成功进入筑基初期事件「筑基后的第一缕灵压」。

### 10.5 地图/位面（轻量）

不做大地图，用 **区域标签** 切换事件池：

| 区域 | 解锁 | 风格 |
|------|------|------|
| 青牛镇/山村 | 开局 | 凡人与炼气 |
| 七玄外门感 | 炼气中 | 门派低语（致敬凡人，不照搬版权名可改「玄门」） |
| 坊市与散修盟 | 炼气后 | 交易、情报 |
| 乱星海边缘 | 结丹+ | 航海、妖兽 |
| 大晋修仙界 | 元婴+ | 高阶政治 |
| 虚天殿类秘境 | 随机 | 一次性高风险副本事件组 |
| 灵界入口 | 化神/炼虚 | 后期 |

（正式文案避免直接照搬小说专有名词侵权风险，用「风格致敬 + 自创地名」。）

## 11. 战斗子系统（选择式）

非即时动作，采用 **回合选择卡**：

1. 展示敌我状态（境界压制会显示「对方高你一整境」）。  
2. 每回合 3–4 行动：攻、守、术、逃、英雄技、用丹。  
3. 敌方 AI 简单策略表（狂攻/控场/召唤）。  
4. 境界差：高一境基础减伤与命中优势（凡人味）。  
5. 战斗结束：掉落、伤势、杀孽、可能的「余波事件」。

MVP 可进一步简化为 **单次综合检定 + 1 个战术选项**，完整版再做多回合。

## 13. 失败、存档与 Roguelike 结构

### 13.1 一局结构（已锁定：纯一命）

- **一命 Roguelike**：身死 / 寿尽 / 道消 → 本局结束，**无复活、无残魂再起、无轮回继承**。  
- **无元进度**：新开一局不携带任何永久数值加成；英雄与事件开局即全部可选（或按内容版本一次性开放，但不靠游玩解锁养成）。  
- **种子**：可分享 `seed` 复现同序事件（竞速/主播/复盘），种子不提供成长收益。  
- **可重玩动力**纯靠：英雄机制差、事件随机、自我挑战更高境界。

### 13.2 死亡原因

- 寿元尽  
- 战斗身死  
- 天劫失败  
- 心魔入体（道心 0）  
- 特殊事件处刑  

死亡后：本局总结（最高境界、关键选择、死因、用时）→ **清除进行中存档** → 回标题。  
胜利同理：通关总结 → 清除进行中存档 → 回标题。

### 13.3 存档（一命友好、防读档重掷）

| 规则 | 说明 |
|------|------|
| 进行中自动档 | 仅 **1 槽**：退出/崩溃后续玩同一条命；**继续游戏**用 |
| 禁止多档 | 无手动 3 槽、无「关键节点存档」 |
| 禁止读档刷 | 选择确认并结算后即写入；**不可**回退到选项前 |
| 死亡/通关 | 删除进行中档；仅可保留只读「上次结局摘要」（可选，无数值） |
| 存储介质 | `localStorage` 或本地 JSON 文件 |

## 18. 关键决策（Key Decisions）

| # | 决策 | 理由 |
|---|------|------|
| K1 | **玩法以事件选择为主，战斗为子系统** | 契合「随机事件+选择」需求，开发量可控 |
| K2 | **境界严格按凡人修仙传大框架** | 用户明确要求；炼气 13 层做新手曲线 |
| K3 | **火炬英雄做「道途」而非数值皮肤** | 用机制差异（怒气/弹药/祝福）保证重玩 |
| K4 | **纯一命，无轮回元进度** | 用户锁定：死亡即终局；重玩只靠随机与英雄差 |
| K5 | **内容数据与逻辑分离（JSON 事件/天赋）** | 扩内容不改代码 |
| K6 | **MVP 只做到结丹，化神为 1.0** | 避免前期内容空洞 |
| K7 | **凡人侧地名/门派自创；英雄用火炬原名** | 境界致敬小说自创地名；角色名保留火炬辨识度 |
| K8 | **Web 技术栈做单机** | 空仓库快速出原型，可再包桌面 |
| K9 | **美术统一火炬暗金** | 用户锁定：暗底金描边，不加水墨皮 |
| K10 | **存档仅 1 槽进行中自动档，结算后不可回退** | 服务一命，防止读档重掷 |
| K11 | **界面与内容仅中文** | 用户锁定；降低工程与文案成本 |
| K12 | **开局随机天赋盘（tlidb Talent 池），局内不可洗** | 用户锁定；用神系+核心+灵纹做每局 Build 差异 |
| K13 | **进度以年历+主动行动驱动** | 修炼/探险/PK 等耗时换修为；满修为进阶；终局渡劫飞升 |
| K14 | **境界对齐凡人：炼气…大乘，终局渡劫期** | 下中境界完整；渡劫为飞升通关阶段 |
| K15 | **功法=火炬主动技能原名** | 池来自 Active_Skill；Tag 保留 |
| K16 | **每大境界阶段 1 个主修** | 新境可重选；往境烙印弱化保留 |
| K17 | **品阶 + 技能等级双轨** | 品阶看获取，等级靠专修与实战 |
| K18 | **战斗/奇遇等获取功法** | 掉落、残篇合书、宗门、坊市；重复转熟练 |

## 21. 成功指标（自测）

- 新玩家 3 分钟内完成第一次有意义的选择。  
- 不同英雄前 20 事件重复率 < 40%。  
- **同英雄不同天赋盘**，前 15 事件可选选项集合有可见差异。  
- 筑基成功率体感：准备充分 ~70%，裸破 ~20%。  
- 一局内至少 1 次「后悔刚才的选择」的叙事反馈。  
- 结丹胜利/身死两种结局均可达成。  
- 玩家能清晰感知「耗时换修为」与寿元压力；渡劫为完整版通关高潮。  
- 每破一大境会重选/确认主修；战斗与奇遇能稳定见到功法掉落反馈。  
- 全 UI 无英文残留（专有名词火炬原名除外）。
